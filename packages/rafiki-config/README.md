# @openbridge/rafiki-config

Rafiki deployment configuration for OpenBridge: environment wiring, the supported
assets and their scales, and ILP peering/seed data. The runtime services
themselves are defined in the repo-root [`docker-compose.yml`](../../docker-compose.yml).

| Corridor | Asset | Scale |
|---|---|---|
| Nigeria | `NGN` | 2 |
| UK | `GBP` | 2 |
| US | `USD` | 2 |

Built for the Interledger Open Payments Accelerator 2026.
