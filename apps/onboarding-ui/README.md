# @openbridge/onboarding-ui

The user-facing surface for OpenBridge:

- Wallet-address creation and management (`$openbridge.org/username`)
- Account linking — including **client-side ZK proving** (raw account details never leave the device)
- The **GNAP interactive-grant consent screens** Rafiki's auth server redirects users to when they authorise an outgoing payment

Built for the Interledger Open Payments Accelerator 2026.
